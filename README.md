# Marine Biology VR

## About

Our entry for [AT&T Developer Summit VR/AR Challenge](https://devsummit.att.com/vr-ar-challenge).  This prototype for a VR "game" aims to teach people about marine biology, by teaching them about fish.

### Technologies Used
- Unity3D
- C#
- git
- GitHub
